# BlockerApp

**BlockerApp** is a lightweight Windows desktop application designed to help you **stay focused** by blocking distracting websites.

Once a website is blocked, it cannot be accessed until you manually unblock it. Perfect for students, developers, or anyone looking to improve their productivity

---

## Features

- Block websites easily
- Unblock websites when needed
- Simple PySide6 GUI
- Compiled with Nuitka for windows

---

## Installation

1. Download the latest installer from the [Releases](https://github.com/draco-ghost/blocker_app/releases) page.
2. Run `BlockerApp_Setup.exe` and follow the installation instruction.
3. Open the app, add websites to block, and enjoy distraction-free time.
